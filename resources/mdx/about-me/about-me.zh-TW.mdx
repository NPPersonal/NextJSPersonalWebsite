### 背景

---

出生於台灣並在昆士蘭科技大學就讀資訊科技. 青年時期對軟體開發非常感興趣,
因此到現在扔然在相關領域不停學習.

由於軟體開包含不同方面的元素, 自己也花幾年時間自我學習不一樣的技術.
無論是程式語言, 框架或開發工具. 只要新技術或知識能協助開發和問題解決, 我都
非常有興趣了解.

過去幾年裡都專注於 IOS 平台原生開發不管是在公司或接案. 由於自己需要一個人網站,
因此也接觸到[React](https://react.dev/)相關的框架. 像是這網站就是由
[Gatsby](https://www.gatsbyjs.com/)框架所建立出來的. 此外行動
裝置跨平台開發開始漸漸嶄露頭角由於它類似於 React 且可以一種語言跨不同平台.因此我
開始學習跨平台開發的相關框架像是[React Native](https://reactnative.dev/). 如上
述因為開發會包含不同方面的元素因此也學習不同的開發工具, 例如 Docker, NPM 等等開發
的協助工具.

近年來機器學習與人工智慧的熱門並且也有越來越多在生活例子上面的應用. 自己也開始學習相關
知識和應用, 並研究如何將人工智慧結合到軟體並改善或提升生活方面的問題. 例如這個[人腦腫瘤
檢測器](https://braintumordetectorapp.streamlit.app/)透過深度學習方式來掃描人腦核磁
共振影像圖並且偵測是否有腫瘤.

### 框架和工具

---

學習新的框架可以增加新的技術和知識, 在往後的開發道路上有機會變成解決問題的方案之一.

以下是我接觸過的框架.

- [IOS SDK](https://en.wikipedia.org/wiki/IOS_SDK): IOS 平台原生開發軟體
- [React.js](https://react.dev/): 一套用於開發網頁前端的開源框架
- [Next.js](https://nextjs.org/): 基於在 React.js 的網頁開發框架, 比 React.js 有更完整功能, 如自行撰寫 Restful API
- [Gatsby](https://www.gatsbyjs.com/): 基於在 React.js 並類似於 Next.js. 強調在廣泛的插件且適合 CMS 的網頁開發. 例如這個
  個人網站就是該框架開發出來的
- [React Native](https://reactnative.dev/): 類似於 React.js 但用於跨行動裝置平台開發
- [Express.js](https://expressjs.com/): 基本在 Node.js 的輕量級框架並用於網頁後端開發, 除此之外具有前端開發的附加功能
- [Tensorflow](https://www.tensorflow.org/): 人工智慧深度學習的框.
- [PyTorch](https://pytorch.org/): 人工智慧深度學習的框架且類似於 Tensorflow. 在學術研究領域更熱門
- [Scikit-Learn](https://scikit-learn.org/stable/): 機器學習的框架
- [Codelgniter](https://www.codeigniter.com/): 輕量級用於 PHP 網頁開發

額外的開發工具和知識有時可以派上用場.
以下是我過去所接觸過工具.

- [Git](https://git-scm.com/book/en/v2/Getting-Started-About-Version-Control): 版本控制工具
- [CI/CD](https://en.wikipedia.org/wiki/CI/CD): 持續整合/持續部屬像是[CircleCI](https://circleci.com/)
  和[TravisCI](https://www.travis-ci.com/)
- [Docker](https://www.docker.com/): 輕量級的虛擬機器但以容器為單位, 以容器為中心和最小單位.
- [Database](https://en.wikipedia.org/wiki/Database): 資料庫例如[MYSQL](https://www.mysql.com/)和
  [Mongodb](https://www.mongodb.com/)
- [CocoaPod](https://cocoapods.org/): IOS 開發的套件管理工具
- [NPM](https://www.npmjs.com/): javascript 相關的套件管理工具
