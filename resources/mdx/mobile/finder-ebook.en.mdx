### Introduction

---

[GOGOFinder](https://www.gogofinder.com.tw/index/index.php) is a company who
provide cloud E-Book platform service for their customer.

This E-Book reader app was designed and worked with the platorm serivce soley.
User is able to use this app to read any books that is provided from the service.

<br />

### Motivation & Purpose

---

This is an app that is builded for IOS mobile device to work with cloud E-Book platform
for company [GOGOFinder](https://www.gogofinder.com.tw/index/index.php).

<br />

### More

---

This IOS app is created from scratch based on previous version with new design
in order to achieve better performance and user experience.

Each published E-book contain different type of media. The app allow reader to
experience these media to increase interaction. For example play an audio
media wthin a book

User is able to read a book while playing audio that is embeded in book for
each pages.

Frameworks were used to build for the app:

- [IOS SDK](https://developer.apple.com/whats-new/): Core framework
- [Testflight](https://testflight.apple.com/): Beta test for client

<br />
